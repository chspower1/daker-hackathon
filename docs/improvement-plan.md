# 개선 계획

현재 버전은 핵심 사용자 흐름에 집중한 프로토타입이며, 아래 항목은 다음 단계 확장 방향이다.

## 1. 현재 범위 안에서 바로 확장할 항목

- 해커톤 목록의 상태/태그 필터 고도화
- 캠프 모집글 수정 및 모집 마감 처리
- 랭킹 기간 필터 추가
- 상태 화면과 빈 상태 메시지의 시나리오별 정교화

## 2. 데이터 모델 보강이 필요한 항목

- 팀 초대, 수락, 거절 흐름
- 제출 이력과 버전 관리
- 해커톤별 더 정밀한 리더보드 정책

## 3. 범위 확장이 필요한 항목

- 인증 및 계정 기반 동기화
- 백엔드 DB 와 쓰기 API 연동
- 관리자용 운영 화면
- 실시간 협업 또는 메시징 기능

## 4. 운영 품질 개선

- 배포 환경 기준의 수동 QA 시나리오 정례화
- 문서와 구현 간 계약 변경 추적 강화
- 제출 흐름과 결과물 관리 체계 정리
