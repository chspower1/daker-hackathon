# 기능 상세: 홈

## 1. 라우트

`/`

## 2. 목적

해커톤 핵심 흐름으로 빠르게 진입시키는 랜딩 허브다.

## 3. 핵심 요소

- CTA 3개 노출
  - `해커톤 보러가기` -> `/hackathons`
  - `팀 찾기` -> `/camp`
  - `랭킹 보기` -> `/rankings`
- 모바일/데스크톱에서 CTA의 의미가 명확해야 함
- 홈은 정적 구현이 가능하며, 읽기 전용 진입점으로 동작함

## 4. 완료 기준

- 3개 핵심 화면을 1클릭으로 진입
- 진입 경로가 모바일에서도 가독성 있게 표시
- 페이지 진입 시 불필요한 에러 상태를 보여주지 않음

## 5. 연계 규칙

홈은 기능군의 시작점이므로 아래 문서의 공통 규칙을 따른다.

- 범위: `docs/common/product-and-scope.md`
- 수용 기준: `docs/common/acceptance.md`
